部署

检查你的脚本文件，确保TTMapV0.0.1包中至少包含以下两个文件：

  • TTMap.js
  • TTMap.min.css

通过服务器发布上述三个静态资源，以至于能够在前端调用他们。你可以使用你熟悉的发布方式: TOMCAT, NODEJS, WebLogic ...

发布成功以后，可以在前端页面中通过脚本引用的方式，把脚本引入到项目中。

HTML

                            
    <!-- 注意加载顺序，文件编码格式为 "utf-8" -->
    <link rel="stylesheet" type="text/css" href="http://{yourip}:{port}/{path}/TTMap.min.css">
    <script src="http://{yourip}:{port}/{path}/TTMap.js"></script>

现在可以使用TTMapV0.0.1的API进行Web地图开发...

新建一张simple地图

如下方式新建simple地图:

HTML

                            
    <!-- 提供地图容器，其中地图容器必须提供id号，用于生成地图 -->
    <div id="map" style="width:1300px;height:600px;"></div>

Javascript

                            
    /**
    * 这里的map为上文HTML中新建的地图容器ID号；
    * 下面一行代码便可以加载地图到browser上
    */  
    var map = new TTMap("map");

完整的HTML代码

HTML

                            
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>Simple Map Demo</title>
        <link rel="stylesheet" type="text/css" href="http://{yourip}:{port}/{path}/TTMap.min.css">
    </head>
    <body>
        <div id="map" style="width: 1300px;height: 600px"></div>
    
        <script type="text/javascript" src="http://{yourip}:{port}/{path}/TTMap.js"></script>
        <!-- 外部js或者内嵌js -->
        <script type="text/javascript">
            var map = new TTMap('map');
        </script>
    </body>
    </html>
